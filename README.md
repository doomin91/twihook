## Description
[Twihook] Twitch Streamer 전용 제공 기능으로서 실시간 채팅의 시각화 및 시청자 참여 컨텐츠를 제공하는 웹 서비스 Twihook입니다.

## 기능구현단계
- [ ] 트위치 연동 로그인
- [ ] 방송 정보 조회
- [ ] 방송 채팅창 조회
- [ ] 욕설 필터링 기능을 통한 밴(Ban) 기능
- [ ] 방송 알람, 앱 푸쉬 기능 ( 현재 ~~ 님이 접속하셨습니다 )
- [ ] 시청자 분석 기능
- [ ] 대쉬보드
- - [ ] 포인트 랭킹
- - [ ] 상품 랭킹
- - [ ] 투표 승/패 랭킹
- - [ ] 채팅 형태 랭킹 ( ㅋㅋㅋ, ㄹㅇㅋㅋ )
- - [ ] 후원 랭킹
- [ ] 카테고리 별 시청자 수 분석, 그래프 + 스냅샷 활용
- [ ] 미션 목록 
- [ ] 트게더 (https://tgd.kr/) 연동
- [ ] 나무위키 연동, 트위치 스트리머 정보
- [ ] 저작권 관련 영상 자동 차단
- [ ] 성인물 관련 영상 자동 차단
